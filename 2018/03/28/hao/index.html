数组的定义：

1.数据类型[] 数组名 = new 数据类型[数组长度]

例：int[] num = new int[5];

表明定义一个整型类型的数组，数组名为num，数组长度为5。

int[] num = new int[]{ 3，4，2，5};

注意：

1.数组下标从0开始，例：a(0),a(1)….

2.java里要给数组分配空间。

数组的使用：

public class Ex1{

public static void main (String [] args){

int [] num= new int[3];

 num[0]=1;

 num[1]=2;

 num[2]=3;

 }

}

注：这段代码表示我们先定义了一个整型类型的数组然后给数组分配了3个空间并给它们赋了值。
